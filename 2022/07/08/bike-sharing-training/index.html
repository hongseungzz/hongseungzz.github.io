Chapter 04. 데이터 시각화

  • 수치를 예측하는 대회
  • 종속변수를 시각화해야 함
  • 분포 확인 후, 로그변환을 줄지 말지를 결정해야 함
  • 시각화를 하는 이유 : 어떤 컬럼을 삭제할 지 고민하기 위해서
  • 막대 그래프
    • year, count
    • month, count
    • day, count
    • hour, count
  • 각 그래프의 타이틀 추가
  • boxplot
    • season, count
    • weather, count
    • holiday, count
    • workingholiday, count
  • 포인트플롯
  • 5개의 행 그래프 작성
    • workingday, holiday, weekday, season, weather
  • 5개의 그래프를 한 이미지로 그리기
  • 회귀선을 포함한 산점도 그래프
  • 총 4개의 그래프가 나와야 함
  • 히트맵 그래프 그리기
  • 상관계수 해석
    • 수치가 양수다 (양의 관계)
    • 수치가 음수다 (음의 관계)
    • 0 ~ +-0.2 : 두 변수 사이의 상관관계는 없다.
    • +-0.2 ~ +-1 : 값이 커지면 커질수록 두 변수 간 상관관계는 크다.
